Introdução ao CSS

O CSS (Cascading Style Sheets) em português Folha de Estilo em Cascatas, é usada para estilizar elementos escritos em uma linguagem de marcação como o HTML.

O CSS separa o conteúdo da representação visual do site. Utilizando o CSS é possível alterar a cor do texto, cor de fundo, fonte e espaçamento entre parágrafo, criar tabelas, usar variações de layouts, ajustar imagens para suas respectivas telas e assim por diante. O CSS foi desenvolvido pelo W3C (World Wide Web Consortium) em 1996.


Existem três tipos de estilos no CSS, sendo:

  • Inline
  • Interno (incorporado)
  • Externo

Estilo Inline

No estilo inline os comandos em CSS ficam dentro das tags do html, através do atributo style. No exemplo a seguir será atribuído a cor blue (azul) ao elemento <h1> através do atributo style.

HTML

<body>
  <h1 style = "color : blue">Conteúdo</h1>
</body>

Estilo Interno

O estilo interno fica na área <head>, onde e aberto às tags <style> </style>, para serem efetuados os comandos.

HTML

<head>
  <title>Introdução ao CSS</title>
  <style>
    h1{
      color : blue
    }
  </style>
</head>

Estilo Externo

No estilo externo e criada uma pasta terminada em .css para ser escritos os comandos.

Para importar o arquivo.css é necessário adicionar na área <head> a tag <link> em conjunto com o atributo href, sendo que o valor do mesmo é a localização do arquivo externo, é definir a relação do arquivo que se pretende importar com o arquivo atual, é a sua função, através do atributo rel e o valo stylesheet (folha de estilo), é necessário ainda definir o tipo do conteúdo usando o atributo type com o valor text/css.

HTML

<head>
  <link rel="stylesheet" type = "text/css" href = "arquivo.css"/>
</head>

Esse estilo é mais recomendado, por quê separar o código HTML do código CSS.

Exemplo completo:

HTML

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <title>Introdução ao CSS</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" type="text/css" href="style.css">
  </head>
  <body>
      <h1>Introdução ao CSS</h1>
  </body>
</html>

CSS

h1{
  color : green;
}
