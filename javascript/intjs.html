Introdução ao Javascript

O Javascript é uma linguagem de programação interpretada, de tipagem fraca e dinâmica, foi criada em 1995 por Brendan Eich.


Mas o que é uma linguagem interpretada?, de tipagem fraca?, é dinâmica?


De forma geral, uma linguagem de programação interpretada é aquela que precisa de um interpretador para que os programas sejam executados, para que em seguida seja executado pelo sistema operacional ou processador.


Uma linguagem de tipagem fraca realiza conversões automaticamente entre os tipos de dados, ou seja, o programador não precisa se preocupar em fazer as conversões de tipo manualmente, a própria linguagem faz isso. Legal né?:)


Já uma linguagem de programação dinâmica e quando determinadas operações podem ser realizadas em tempo de execução, ou seja, é possível mudar o tipo de uma variável, adicionar novos métodos ou propriedades a um objeto enquanto o programa está sendo executado.


Agora que você já entendeu as principais características do javascript, é hora de aprender como adicionar ele no seu código HTML.

O código javascript pode ser colocado nas áreas <head> </head> ou <body> </body>, ou em ambas, através da tag <script> </script>.

No exemplo a seguir vamos colocar o código javascript na área <head> </head>, e criar um script que irá mostrar uma mensagem com a frase "Hello, World" na tela, usando o método alert().

EXEMPLO

<!DOCTYPE html>
<html lang="pt-br">
 <head>
    <title>Introdução ao Javascript</title>
    <meta charset="UTF-8">
    <script>
      alert("Hello, World");
    </script>
  </head>
  <body>
   <h1>Introdução ao Javascript</h1>
 </body>
</html>

Agora vamos colocar o mesmo script na área <body> </body>.

EXEMPLO

<!DOCTYPE html>
<html lang="pt-br">
 <head>
    <title>Introdução ao Javascript</title>
    <meta charset="UTF-8">
  </head>
  <body>
   <h1>Introdução ao Javascript</h1>
   <script>
      alert("Hello, World");
    </script>
  </body>
</html>

Há principal diferença em colocar o código javascript na área <head> ou na <body> e o fato de que os scripts na parte inferior da área <body> melhora a velocidade de carregamento da página web.

O código javascript também pode ser colocado em arquivos externos.

Para usar o código javascript em um arquivo externo, deve-se criar um arquivo com a extensão ".js", como por exemplo "script.js", é logo após adicionar a localização do arquivo javascript (script.js) na área <head> ou na <body>, usando a tag <script> </script> em conjunto com o atributo "src".

EXEMPLO

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <title>Introdução ao Javascript</title>
    <meta charset="UTF-8">
  </head>
  <body>
   <h1>Introdução ao Javascript</h1>
    <script src="script.js"></script>
  </body>
</html>
