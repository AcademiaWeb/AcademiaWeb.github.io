Introdução ao HTML5

De forma geral o HTML (Hyper Text Markup Language), é uma Linguagem de Marcação de Hipertexto, usada na construção de páginas Web.

O HTML foi desenvolvido por Tim Berners-Lee, em 1991 no CERN (European Council for Nuclear Research) na suíça.

A estrutura básica do HTML é formada por tags, elementos e atributos.


As Tags são um conjunto de caracteres que formam um elemento, indicando o seu começo e fim.

Existem dois tipos de tags, sendo as:


Que precisam de fechamento:

HTML

<body>
  <p>Conteúdo</p>
</body>

É as que não presisam ser fechadas:

HTML

<body>
  Conteúdo<br>
</body>

Os Elementos são compostos pelas tags e o conteúdo contido dentro da mesma, conteúdo esse que pode ser imagem, texto, elemento vazio.


HTML

<body>
  Conteúdo<br>
</body>

O exemplo acima é de um elemento vazio, pois é uma quebra de linha, ou seja o elemento da tag e um espaço.


Os atributos são informações complementares, que deve ser atribuídos a uma tag para que a mesma se comporte de uma determinada forma. O exemplo a seguir atribui a tag <html> a função de informa a linguagem predominante no texto.


HTML

<html lang="pt-br">
  <title>Teste</title>
</html>

No exemplo a seguir está a estrutura básica do código html:


HTML

<!DOCTYPE html>
<html lang = "pt-br">
<head>
  <title> Título da página Web</title>
  <meta charset = "UTF-8">
</head>
<body>
  Conteúdo da página web
</body>
</html>

Definição:

<!DOCTYPE html> : Informa ao browser (navegador) qual é o tipo do documento, é em que versão está. No caso ao colocá somente HTML no final o navegador entende que está na versão 5. (Não é uma tag e uma instrução para o navegador.)


<html lang = "pt-br"> </html> : Indica o início e o fim do código, é informa através do atributo lang qual é a linguagem predominante. No caso é o português Brasil.


<head> </head> : Nesta área fica o cabeçalho do código, ou seja é nesta área onde será definido qual conjunto de símbolos o texto usa, título, palavras-chave, etc.


<title> </title> : Tag para definir o título da página Web, o mesmo é exibido na aba da página.


<meta charset = "UTF-8"> : Está tag serve para especificar qual conjunto de acentos / símbolos a página web está usando, no caso o "UTF-8" equivale ao conjunto de acentos / símbolos usado no português Brasil.


<body> </body> : É nesta área onde fica o conteúdo da página Web.
